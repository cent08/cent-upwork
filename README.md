# Cent Robles - Upwork Portfolio

Upwork-facing version of my portfolio. Same design and project showcase as the
main site, with all direct contact details and booking removed. Every project
and hire request goes through my Upwork profile.

Static site: HTML + vanilla JS + CSS, no build step.

- `index.html` - markup and styles
- `app.js` - interactions (carousel, reveals, flow-field background)
- `assets/`, `images/` - media

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```
